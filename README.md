# Traumferienwohnung Manager

Modernes Verwaltungstool für Ferienwohnungen mit Kalender, Statistiken und Multi-User-Support.

## Features

- 📅 Kalender-Management (14-Tage-View, smooth scroll)
- 👥 Multi-User (bis zu 5 Konten mit Email-Verifikation)
- 📊 Statistiken & Auslastung
- ⭐ Wiederkehrende Gäste (Stern bei 2+ Buchungen)
- 💬 Notizen-System pro Buchung
- 🔐 Sichere Authentication

## Unterstützte Wohnungen

**Hus Upstalsboom:** Wohnungen 1-6  
**Haus Anne:** Wohnungen 1-5

## Tech Stack

- Frontend: React + Tailwind CSS
- Datenbank: Firebase
- Hosting: Vercel (Frontend)
